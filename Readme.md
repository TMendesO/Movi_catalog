# Movie Catalog API

API RESTful para gerenciar um catálogo de filmes.

## Pré-requisitos

- Node.js
- Docker
- Docker Compose

## Instalação

```bash
# Instalar as dependências
npm install

# Iniciar o PostgreSQL e Redis com Docker Compose
docker-compose up -d
```
